# Github Kanban board

## Stack

- TypeScript
- ReactJS
- Redux Toolkit
- Ant Design
- React Testing Library

## Deployment

Create the .env file. Add github key:

```bash
  REACT_APP_GITHUB_KEY=
```

Install all packages, using commands:

```bash
  npm install
```

Then, run project, using this command:

```bash
  npm start
```
